# SARS-CoV-2 nsp14 Variant Analysis

## Project Overview
This project performs a comparative bioinformatics analysis of the SARS-CoV-2 non-structural protein 14 (nsp14) from Wuhan, Delta, and Omicron variants.

## Objectives
- Extract nsp14 protein sequences from ORF1ab
- Compare amino acid composition across variants
- Identify mutations relative to Wuhan strain
- Perform pairwise alignment and sequence comparison

## Tools & Technologies
- Python
- Biopython
- Pandas
- Matplotlib
- VS Code

## Key Analyses
- FASTA sequence processing
- Amino acid composition analysis
- Variant-wise mutation mapping
- Alignment using Biopython PairwiseAligner

## Results
- Variant-specific and conserved mutations identified in nsp14
- Delta and Omicron show distinct evolutionary changes
